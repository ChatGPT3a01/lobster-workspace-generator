## 📖 簡介

這是一個 **線上 Workspace 模板生成器**，專為 [OpenClaw](https://github.com/nicholasgasior/openclaw) 個人 AI 助理設計。

只要在網頁上填寫設定，就能一鍵生成 6 個核心設定檔，讓你的龍蝦 AI 助理擁有獨一無二的世界觀與人格。

## 🚀 快速開始

```
你想怎麼使用？
│
├─► 線上使用（推薦）──► 點擊上方 GitHub Pages 連結
│
└─► 本地使用 ─────────► 下載 index.html + logo.png，用瀏覽器開啟
```

### 使用步驟

1. 開啟[線上生成器](https://chatgpt3a01.github.io/lobster-workspace-generator/)
2. 依照 6 個步驟填寫你的 AI 助理設定
3. 點擊「預覽」查看生成結果
4. 滿意後點擊「下載全部」取得所有檔案
5. 將下載的檔案放入 OpenClaw 的 `workspace/` 資料夾

---

## 📁 Workspace 結構說明

```
workspace/
├── IDENTITY.md    ← 你的 AI 助理「身份證」
├── SOUL.md        ← 行為準則與核心價值觀
├── USER.md        ← 關於你（使用者）的資訊
├── AGENTS.md      ← Agent 記憶、安全、通訊規範
├── TOOLS.md       ← 工具使用注意事項
└── HEARTBEAT.md   ← 排程與心跳提醒設定
```

📖 使用教學

這是什麼？

此工具幫你生成 OpenClaw 龍蝦AI 的 Workspace 人格模板。OpenClaw 是一個開源的個人 AI 助理框架，透過 workspace/ 資料夾中的 .md 檔案來定義 AI 的人格、行為和能力。

這個生成器會幫你產出 6 個核心設定檔，下載後放進 workspace 目錄，你的龍蝦 AI 就能擁有你設計的人格。

⚠️ 前提：使用此工具前，你需要先安裝好 OpenClaw。如果還沒安裝，請先使用「龍蝦自動安裝程式」完成安裝（含 Node.js、OpenClaw、LINE 插件等）。

快速使用步驟

1填寫設定 — 點下方 6 個 Tab（身份→靈魂→使用者→代理人→工具→心跳），依序填寫。也可點「載入範例」快速體驗。
2生成龍蝦 — 按下底部的「🦞 生成我的龍蝦」按鈕。
3下載檔案 — 在彈出的人格名片上點擊綠色「📦 下載全部檔案」按鈕，6 個 .md 檔會自動下載。
4放入 Workspace — 把下載的檔案放到你的 OpenClaw workspace 目錄（見下方路徑）。
5重啟 OpenClaw — 執行 openclaw gateway restart，龍蝦就會載入新的人格設定。

檔案放置位置

Windows：C:\Users\你的使用者名稱\.openclaw\workspace\
Mac/Linux：~/.openclaw/workspace/

workspace/
├── IDENTITY.md    ← 龍蝦的名字和角色
├── SOUL.md        ← 行為準則和靈魂
├── USER.md        ← 你（使用者）的資訊
├── AGENTS.md      ← 記憶系統和安全規則
├── TOOLS.md       ← 工具偏好和環境設定
├── HEARTBEAT.md   ← 定期檢查排程
├── memory/        ← 記憶系統（自動產生）
│   └── YYYY-MM-DD.md
└── skills/        ← 技能包目錄
    └── (由 clawdhub 安裝)
⚠️ 如果 workspace 已有這些檔案，下載放入會覆蓋原有內容。建議先備份：
copy *.md backup/

6 個設定檔詳細說明

🦞 IDENTITY.md — 身份設定（Tab：身份）
龍蝦的「身份證」。定義名字、角色、個性風格和代表 Emoji。
  • 龍蝦名稱 — 你想叫 AI 什麼名字（例如「亮的龍蝦」）
  • 角色定位 — 龍蝦在你生活中的角色（例如「個人助理」「程式碼審查員」）
  • 個性風格 — 說話的感覺，可用快選卡片一鍵帶入
  • 代表 Emoji — 龍蝦在對話中使用的代表 emoji
💫 SOUL.md — 靈魂設定（Tab：靈魂）
龍蝦最重要的檔案！定義「價值觀」和行為準則。
  • 核心行為準則 — 5 個預設準則，勾選你想要的（建議全勾）
  • 自訂準則 — 加入你獨有的規則（例如「回答時優先用中文」）
  • 溝通風格 — 描述龍蝦應該怎麼跟你聊天
  • 邊界規則 — 龍蝦不應該做的事（隱私、群組行為等）

修改 SOUL.md 等於改了龍蝦的「核心」，請想清楚再改。

👤 USER.md — 使用者資訊（Tab：使用者）
告訴龍蝦「你是誰」，讓它更了解你。
  • 名字 / 稱呼 — 龍蝦怎麼叫你（可以設定暱稱）
  • 時區 — 影響行事曆提醒、安靜時段等時間相關功能
  • 主要語言 — 龍蝦回覆你的語言偏好
  • 個人背景 — 你的職業、興趣、常用工具等，幫龍蝦更好地輔助你
🤖 AGENTS.md — 代理人規範（Tab：代理人）
龍蝦每次啟動時的行為準則和工作流程。
  • 記憶系統 — 每日記憶（memory/YYYY-MM-DD.md）和長期記憶（MEMORY.md）。建議全部開啟，龍蝦才能記住跟你的對話脈絡
  • 安全原則 — 防止龍蝦執行危險操作（刪除檔案、外洩隱私等）
  • 群組聊天規則 — 如果你在 Discord / LINE 群組使用龍蝦，這些規則決定它何時該發言、何時安靜
  • 語音說故事 — 需要安裝 ElevenLabs TTS（sag 技能），龍蝦會用語音朗讀故事和摘要
  • 平台格式適配 — 龍蝦會根據平台（Discord/WhatsApp/LINE）自動調整訊息格式
🔧 TOOLS.md — 工具筆記（Tab：工具）
龍蝦的「小抄」，記錄你的環境特有設定。
  • TTS 語音偏好 — 龍蝦透過 LINE 發語音訊息時的聲音風格
  • 自訂環境筆記 — 你的 SSH 伺服器、常用 IDE、裝置名稱等。龍蝦會參考這些資訊來協助你
💓 HEARTBEAT.md — 心跳排程（Tab：心跳）
讓龍蝦「主動」定期檢查任務。
  • 不啟用 — 檔案留空/註解，不會產生 API 呼叫，省錢
  • 輕量模式 — 只檢查行事曆和天氣（每天 2~4 次）
  • 完整模式 — Email + 行事曆 + 天氣 + 社群 + 專案，全部檢查
  • 自訂 — 自己勾選要檢查的項目

每次心跳都會消耗 API 額度。如果是付費 API（如 OpenAI），不需要就選「不啟用」。

TTS 語音功能設定

龍蝦可以透過語音跟你互動（例如在 LINE 上發語音訊息）。有兩種 TTS 方案：

方案 A：Sherpa-onnx TTS（本地離線，免費）
  • 不需要雲端 API Key，在你的電腦上離線運行
  • 安裝方式：使用龍蝦自動安裝程式 → 全部安裝時會自動安裝
  • 模型路徑：~/.openclaw/tools/sherpa-onnx-tts/models/
  • 環境變數（安裝程式會自動設定）：
    SHERPA_ONNX_RUNTIME_DIR=~/.openclaw/tools/sherpa-onnx-tts/runtime SHERPA_ONNX_MODEL_DIR=~/.openclaw/tools/sherpa-onnx-tts/models/vits-piper-en_US-lessac-high
  • 適合：不想花錢、英語語音為主的使用者
方案 B：ElevenLabs TTS（雲端，付費/有限免費）
  • 高品質多語言語音，適合中文語音
  • 需要 ElevenLabs API Key：到 elevenlabs.io 註冊取得
  • 在 AGENTS.md 勾選「語音說故事模式」來啟用
  • 對應 OpenClaw 的 sag 技能
  • 適合：想要高品質中文語音的使用者

在此生成器的「工具」Tab 中選擇的 TTS 語音偏好，會寫入 TOOLS.md，龍蝦會以此作為預設語音風格。

Whisper 語音轉文字設定

龍蝦可以聽懂你的語音訊息，將語音轉成文字再回覆。有兩種方案：

方案 A：本地 Whisper（免費，需要較好的電腦）
  • 不需要 API Key，在本機執行
  • 安裝：pip install openai-whisper
  • 模型會自動下載到 ~/.cache/whisper
  • 首次使用建議用 medium 模型（中英文都不錯）
  • 需要較好的 GPU 或耐心等待（CPU 也可以，但較慢）
方案 B：OpenAI Whisper API（付費，速度快）
  • 需要 OpenAI API Key
  • 取得方式：到 platform.openai.com/api-keys 建立
  • 設定位置：~/.openclaw/openclaw.json 中的：
    "skills" → "entries" → "openai-whisper-api" → "apiKey": "sk-proj-你的金鑰"
  • 或使用龍蝦安裝程式選單 [K] API Key 管理 來設定
  • 費用：約 $0.006 / 分鐘（很便宜）

你可能需要的 API Key（附取得教學）

以下每個 Key 都有詳細的取得步驟。設定方式統一為：使用龍蝦安裝程式選單 [K] API Key 管理，或手動編輯 ~/.openclaw/openclaw.json。

🟢 LINE Channel Secret & Access Token（必要 — 如果用 LINE）

讓龍蝦能透過 LINE 跟你對話。

  1. 前往 LINE Developers Console（用 LINE 帳號登入）
  2. 建立一個 Provider（名稱隨意，例如「我的龍蝦」）
  3. 建立一個 Messaging API Channel
  4. 在 Basic settings 頁面找到 Channel secret → 複製
  5. 切到 Messaging API 頁面 → 最下方 Channel access token → 點 Issue → 複製
  6. 將兩個值貼入龍蝦安裝程式的設定中

費用：免費（LINE Messaging API 免費方案每月可發 200 則推播）

🟢 ngrok Authtoken（必要 — 本地開發用）

讓外部服務（LINE）能找到你電腦上的龍蝦。

  1. 前往 ngrok.com 註冊免費帳號
  2. 登入後，在左側選單找到 Your Authtoken
  3. 複製 token（長得像 2abc123def456...）
  4. 貼入龍蝦安裝程式選單 [7] 設定 ngrok Authtoken

費用：免費方案足夠個人使用（每次重啟 URL 會變，需重新設定 LINE Webhook）

🔵 OpenAI API Key（選用 — AI 模型之一，付費）

使用 ChatGPT 系列模型作為龍蝦的大腦。也用於 Whisper 語音轉文字 API。

  1. 前往 platform.openai.com/api-keys（需註冊 OpenAI 帳號）
  2. 點 Create new secret key
  3. 命名（例如「龍蝦AI」）→ 點 Create
  4. 立刻複製！（只會顯示一次，之後無法再看到）
  5. 金鑰格式：sk-proj-xxxxxxxx...

費用：按使用量計費。GPT-4o 約 $2.5~10/百萬 token。首次註冊有 $5 免費額度。Whisper API 約 $0.006/分鐘。

🔵 Google AI API Key（選用 — 免費 Gemini 模型，推薦新手）

使用 Google Gemini 模型作為龍蝦的大腦，有免費額度。

  1. 前往 Google AI Studio（用 Google 帳號登入）
  2. 點 Create API Key
  3. 選擇一個 Google Cloud 專案（或建立新的）
  4. 複製產生的 API Key
  5. 金鑰格式：AIzaSy...

費用：Gemini 2.5 Flash 免費方案每分鐘 15 次請求、每天 1500 次。非常適合入門！

🟡 ElevenLabs API Key（選用 — 高品質語音 TTS）

讓龍蝦用高品質人聲說話（支援中文），用於語音說故事等功能。

  1. 前往 elevenlabs.io 註冊帳號
  2. 登入後，點右上角頭像 → Profile + API key
  3. 找到 API Key 區塊 → 點眼睛圖示顯示 → 複製
  4. 金鑰格式：sk_xxxxxxxx...

費用：免費方案每月 10,000 字元（約 10 分鐘語音）。付費方案 $5/月起。

⚠️ AI 模型至少選一個：OpenAI（付費）或 Google AI（免費額度）。建議新手先用 Google AI（Gemini），免費就夠用。進階使用者可以兩個都設定，龍蝦會根據需要選用。

常用 OpenClaw 指令

# 啟動/停止/重啟
openclaw gateway start
openclaw gateway stop
openclaw gateway restart

# 健康檢查
openclaw doctor

# LINE 配對
openclaw pairing approve line <配對碼>

# 安裝技能
clawdhub search "關鍵字"
clawdhub install 技能名稱

# ngrok 通道（讓 LINE 找到你的電腦）
ngrok http 18789

小提示

  • 這些設定檔只是「起點」，龍蝦的人格會隨著使用逐漸演化
  • 你隨時可以用文字編輯器（記事本、VS Code）手動修改 .md 檔
  • SOUL.md 是龍蝦的「靈魂」，修改前建議先想清楚
  • HEARTBEAT.md 留空 = 不啟用 = 不消耗 API 額度
  • 記憶系統的 memory/ 資料夾會由龍蝦自動建立，不需要手動建立
  • 如果遇到問題，可以用 openclaw doctor 做健康檢查
